# 通用大模型平台接入文档

版本更新：V2.0

更新时间：2025.05.23

更新内容：1.1小节-请求参数更新，增加多模态、绿网等级等参数

## 1. 大模型Chat

### 1.1 请求路径

| 域名 | https://ai-chat.hep.com.cn |
| --- | --- |
| 路径 | /llm/chat/completions |
| 类型 | POST |

### 1.2 请求Header

| 属性名 | 类型 | 说明 |
| --- | --- | --- |
| system-token | string | 调用llm时的密钥，由管理员分配 |
| Content-Type | application/json |  |

### 1.3 请求参数

| 字段 | 字段 | 字段 | 类型 | 是否必填 | 说明 | 描述 |
| --- | --- | --- | --- | --- | --- | --- |
| messages |  |  | List\<Message\> | Y | 消息列表 |  |
|  |  | role | String | Y | 角色 |  |
|  |  | content | String | Y | 对话内容 | 模型是Qwen-vl-max时：content内容见1.4 |
|  |  | reasoning_content | String |  | 深度思考内容 |  |
| botId |  |  | String |  | 应用ID |  |
| model |  |  | String |  | 模型类型 |  |
| modelCode |  |  | Integer | Y | 模型ID | 盘古：6001；盘古（32K）：6002；三乐：8001；OpenTrek-72B：12003；deepseek_r1: 80001；deepseek_v3: 80002；Qwen: 10002；Qwen-Long: 10008；Qwen-vl-max: 10005 |
| stream |  |  | boolean | Y | 是否流式返回 | 文生图时必须传false |
| advanced_config |  |  |  |  |  |  |
|  | summary_config |  |  |  |  |  |
|  |  | max_tokens | integer |  | 生成的最大token数 |  |
|  |  | tempernature | long |  |  | 0.1~1.0，温度参数，越高，内容随机性越强，创造力越好（推荐使用0.51~1.0） |
|  |  | top_p | double |  |  | 0.1~1.0，限制从概率累加靠前的top_p个结果中获取，一般默认为0.3 |
|  |  | top_k | integer |  |  | 整数，限制从概率靠前的top_k个结果中获取 |
|  | multiModeConfig |  |  |  |  |  |
|  |  | n | integer |  |  | 文生图使用，生成图片的数量。最多4张 |
|  |  | size | String |  | 生成文件大小 | 文生图使用，如：1024\*1024。文生图支持尺寸：512\*512、1024\*1024、1024\*768、768\*1024、720\*1280和1280\*720像素，默认1024\*1024像素。 |
|  |  | prompt_extend | Boolean |  |  |  |
|  |  | watermark | Boolean |  |  |  |
|  |  | negative_prompt | String |  |  |  |
|  |  | widthPx | Integer |  | 分辨率x |  |
|  |  | heightPx | Integer |  | 分辨率y |  |
|  |  | voiceModel | Integer |  |  |  |
|  | csiLevel |  | String |  | 是否走绿网审核 | 默认走普通审核模式，normal-普通审核模式，none-不执行审核 |
